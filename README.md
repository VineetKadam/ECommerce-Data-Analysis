# End-to-End E-Commerce Data Pipeline & Analytics System

## 📌 Overview

This project builds a complete data pipeline to process and analyze e-commerce sales data, enabling business insights and data-driven decision making.

---

## 🚀 Features

* Data cleaning and preprocessing using Python (Pandas)
* Batch data pipeline for transforming raw data into structured format
* Data storage and querying using PostgreSQL
* SQL-based analysis for business insights
* Interactive dashboards built using Tableau
* Customer segmentation using KMeans clustering

---

## 🛠️ Tech Stack

* Python (Pandas, NumPy, Scikit-learn)
* SQL (PostgreSQL)
* Tableau

---

## 📊 Key Insights

* Identified top-performing regions contributing to revenue
* Analyzed monthly sales trends for demand patterns
* Detected top products driving business performance
* Segmented data into low, medium, and high-value clusters
